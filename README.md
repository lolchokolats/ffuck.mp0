## 🎯 **Что это такое?**

**ffuck.mp0** — это **GUI-рекордер** на Python + Tkinter + FFmpeg, который:

- **Записывает экран** в H.264/AAC (MKV).
- **Разбивает на куски** по таймеру (2 мин по умолчанию).
- **Пауза/продолжить** — через SIGSTOP/SIGCONT.
- **Авто-имя**: `video1.mkv`, `video2.mkv`, `video3.mkv...`.
- **Выбор размера**: Full HD, 720p, 480p и др.
- **Выбор звука**: PulseAudio.
- **Темы**: светлая/тёмная/синяя.

## 🎮 **Как пользоваться (30 секунд)**

```
1. Запусти ffuck.mp0
2. Имя файла: "video" (или как хочешь)
3. Папка: Обзор...
4. Размер: 1280x720 (или другой)
5. "Запись по кускам": включи (info объяснит)
6. СТАРТ → ПАУЗА → СТОП
```

## 🔧 **Кастомизация**

**В коде**:
```
-preset ultrafast  →  slower (лучше качество)
-crf 28           →  23 (лучше), 32 (хуже)
-framerate 30     →  60
```

**Аудио**:
```bash
pactl list sources short
```

---

## 🐛 **Возможные проблемы**

**❌ Нет звука**  
`pactl list sources` → выбери правильный.

**❌ Не записывает**  
`ffmpeg` должен быть в PATH.
